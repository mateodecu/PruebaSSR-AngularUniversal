Bienvenido a Star Wars App

Se trata de una simple aplicación Angular que utiliza la API de información sobre Star Wars (SWAPI) para mostrar contenido y para ensayar el uso de Angular Universal para SSR

Para poder testear la renderización de Open Graph, se recomienda que instale la extension Localhost Open Graph Checker en Google Chrome

Respecto de la aplicacion y prueba de Angular Universal, se siguen los pasos del tutorial oficial de Angular

Una vez creado del server-side module mediante el comando  ng add @nguniversal/express-engine  se procede a la implementacion de los Open Graph mediante el modelo planteado en el siguiente tutorial
